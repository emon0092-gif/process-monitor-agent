# Windows Service & Process Monitoring Agent

## Project Description
This project is a cybersecurity monitoring tool that detects suspicious processes, services, and abnormal behavior in Windows systems.

## Features
- Parent-child process monitoring
- Unauthorized process detection
- Suspicious service detection
- Real-time alert generation
- Logging system

## Technologies Used
- Python
- psutil
- wmi
- pywin32

## How to Run
```bash
pip install -r requirements.txt
python main.py
